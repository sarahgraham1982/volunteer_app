# Volunteer App 

An app that incentivizes volunteers with tangiible rewards for voluntary tasks carried out.

## Getting Started

Instructions for how to get up and running...

### Server

Install server dependencies:

```
cd server
npm install
```

Seed the database:

```
npm run seeds
```

Run express:

```
npm run server:dev
```

### Client

```
cd ../client
npm install
npm install react-router-dom
npm start
```
